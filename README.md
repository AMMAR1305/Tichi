# Tichi Web Application - Manual Testing

## Project Overview

This repository contains the **Manual Testing** documentation for the **Tichi Web Application**. The project focuses on validating the application's functionality, usability, security, and responsiveness through comprehensive manual test cases and bug reporting.

---

## Project Scope

The following modules were tested:

- Login
- Signup
- Continue with Google Authentication
- Forgot Password
- Password Validation
- Responsive User Interface
- Input Validation
- Session Handling
- Security Testing

---
